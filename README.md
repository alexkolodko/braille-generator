# braille-app

Based on:

* jsPDF
* svg2pdf.js
* [BrailleToSVG](https://zwettekop.github.io/BrailleToSVG/)



## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
